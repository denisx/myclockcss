Инструкция

myClockCSS - прототип CSS3 часов без использования графики и JavaScript. Текущее время получается с сервера.

Прототип появился по наводке Andreyka Lechev.

Тег <b> встречется в коде 138 раз.

Работает только в браузерах Chrome и Mozilla.

Ещё есть прототип JavaScript часов myClockJS.
